Un poquito de mi

Desde el primer momento en que me sumergí en el mundo de la tecnología, descubrí una pasión inquebrantable por el desarrollo de software. Soy un Desarrollador FullStack que comenzó su carrera profesional motivado por la necesidad. Sin saber que estaba entrando en un campo altamente demandado y valorado, inicié mi trayectoria como Gestor de Bases de Datos. A medida que las necesidades surgieron, fui escalando y adquiriendo nuevas habilidades. Mi formación ha sido principalmente autodidacta, nutriéndome de recursos en línea, prácticas intensivas y experiencia laboral directa. A lo largo de mi carrera, he adquirido un profundo conocimiento en metodologías ágiles, así como en diversas herramientas de DevOps, lo que me permite integrarme eficientemente en equipos y proyectos modernos. Esta combinación de aprendizaje continuo y práctica me ha permitido desarrollar una sólida competencia en el desarrollo full stack.

Mis Experiencias

Assurant

A pesar de que el puesto era para desarrollo Backend, no dudé en desplegar mis habilidades Full Stack, lo que me permitió participar en el desarrollo y diseño de aplicaciones web y móviles utilizando tecnologías como React, React Native y Next, tanto en JavaScript como en TypeScript. También me nutrí de la experiencia en el desarrollo con versiones de .NET como .NET 7, utilizando herramientas como Entity Framework y Entity Framework Core. El stack tecnológico utilizado para DevOps también variaba entre Microsoft Azure y GitHub. Tuve la oportunidad de realizar despliegues desde VS Code en Azure y desde Visual Studio 2022. Cabe destacar que antes de ingresar, no tenía noción acerca de herramientas como React o React Native, lo que me llevó a aprenderlas al punto de desarrollar proyectos personales. Por último, tuve la oportunidad de familiarizarme con herramientas de integración como OKTA.
El stack tecnológico utilizado era: C# .NET (en general), SQL Server, JavaScript (React, React Native, Next) y TypeScript (React y Next).

Aun así, me adapté rápidamente y acompañé a cada uno en ese viaje que me llenó de experiencias de todo tipo. Comprendí las metodologías ágiles y me familiaricé con el stack tecnológico, interiorizándome con herramientas muy comunes en el mercado como SQL Server y MVC. Casa Kindebaluc

Al igual que en NeuralTech, este lugar era modesto y también me brindó la oportunidad de nutrirme de experiencia. Aquí fue donde mi carrera como desarrollador FullStack comenzó. Inicialmente, era un simple gestor de bases de datos MySQL, donde pude crear bases de datos y procedimientos almacenados. Por cuestiones de necesidad, continué como programador en este lugar. Empecé con Visual Fox Pro 9 a asentar las primeras bases, para luego tomar la iniciativa de migrar algunas partes del sistema principal a tecnologías más modernas y escalables, como C#, JavaScript, HTML y CSS. Me encargaba de desarrollar aplicaciones de escritorio tanto en Visual Fox Pro 9 como en WinForms, utilizando Node.js para conectar una base de datos MySQL. Este fue el primer trabajo en el que usé Express.js, y a partir de ahí comencé a trazar mi rumbo como desarrollador. No había metodologías ágiles y yo era el único desarrollador que quedaba, pero aun así me planteé mis propias exigencias laborales.
El stack tecnológico utilizado era: Visual Fox Pro 9, C# (WinForms), JavaScript (Node.js, Express), HTML, CSS y MySQL.

Drive Up

Aquí es donde comienza mi historia. Después de mucho esfuerzo, logré conseguir un puesto como instalador en esta empresa de telemetría, donde me ocupaba de tareas sencillas como el mantenimiento y limpieza de módems, su configuración utilizando la herramienta Putty, y registrando todo en Trello. La pieza clave fue la base de datos PostgreSQL. Me di cuenta de que tenía afinidad para aprender SQL, que se me daba bien y que me gustaba bastante escribir código en este lenguaje. Esto fue fundamental para iniciar mi carrera como gestor de bases de datos y, posteriormente, como desarrollador. Mis labores diarias incluían la limpieza y mantenimiento de los equipos, flashear cada módem, registrarlos en Jasper (Claro), controlar la web principal y comunicar errores, configurar los módems a distancia y manejar la base de datos.
Este trabajo fue muy importante para mí, ya que fue mi primer empleo.
El stack tecnológico utilizado era: Putty, Trello y Jasper (Claro).
